### What is a curried function?

---


<div style={{ width: '80%', fontSize: '.75em'}}>
It's a function with multiple parameters, that is split into smaller, unary (single parameter) functions
</div>

---

<CodeSurferColumns themes={[github]}>

<Step>

```js title="Uncurried"

const getProp = (prop, obj) => obj[prop]

getProp('id', {id: 2}) // 2

```

```js title="Curried"

const getProp = prop => obj => obj[prop]

const getId = getProp('id') // obj => obj['id']

getId({id: 2})

```

</Step>

</CodeSurferColumns>
